== Description ==

Ivaldi Mail Collector allows you to collect email addresses of users visiting your website using a widget. The email addresses can then be copied to your email program to sent them a news letter for example.

This plugin is currently available in these languages: English, French and Dutch. If you create a translation, please send it to us so we can include it in the next release.

== Installation ==

1. Upload the ivaldi-mail-collector directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the widget to one of your sidebars.
